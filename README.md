PM Suryoday Yojana Web Application

This is a web application designed to provide information and facilitate inquiries about the PM Suryoday Yojana, an initiative by the government to promote the use of solar energy. The application includes a user-friendly interface for navigating through various sections such as services, company information, legal policies, and an enquiry form.
Features
###############################  The Project in working Stage ####################################
    Responsive Navigation Bar: A sticky navigation bar that adapts to screen sizes, offering quick access to different sections of the site.
    Informative Banner: A visually appealing banner that highlights the main features and benefits of the PM Suryoday Yojana.
    Detailed Information Section: Provides comprehensive details about the benefits of solar energy, application processes, and government incentives.
    Enquiry Form: Allows users to submit their contact details and queries, ensuring they receive timely responses.
    Footer: Contains links to services, company information, legal policies, and a newsletter subscription form.

Components

    NavBar: A navigation bar with links to various sections, including a search bar and theme toggle.
    Banner: A section that displays a promotional banner with key information.
    Info: Detailed information about the PM Suryoday Yojana, benefits of solar energy, and application processes.
    Enquiry: A form for users to submit their contact details and queries.
    Footer: Contains links to various pages and a newsletter subscription form.

How to Use

    Clone the Repository:

    sh

git clone https://github.com/your-username/pm-suryoday-yojana.git
cd pm-suryoday-yojana

Install Dependencies:

sh

npm install

Run the Application:

sh

Navigate the Application:
Open your web browser and go to http://localhost:3000 to view the application.
